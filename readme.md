# 项目背景

很多用户在日常中都有下载保存全网各平台视频的需求，但受限于平台规则，往往无法直接下载或批量处理。

本项目旨在开发一个支持多平台、多端随时访问的视频下载网站，合理利用 AI 工具和顶级开源项目，低成本、高效率地落地实际商业想法。

## AI 工具栈

- **AI 编程枢纽**：Cursor（自带 Browser Use 浏览器自动化测试验证）
- **AI 能力扩展**
  - **MCP（模型上下文协议）**：通过 Firecrawl 自动抓取网页与开源资料，利用 Context7 获取最新鲜的技术文档，防止 AI 写出过时代码
  - **Agent Skills**：引入 ui-ux-pro-max 技能，让 AI 摆脱千篇一律的设计，生成极具商业感和科技感的前端界面
- **业务架构**
  - **前端**：Vue 3 + Vite + Tailwind CSS
  - **后端**：Python + FastAPI（采用无数据库的轻量级设计）
  - **核心引擎**：yt-dlp（GitHub 斩获 14w+ Star 的视频解析下载神器）

## 核心功能

- **全平台视频无缝解析**：利用 yt-dlp 强大的底层能力，配合自定义脚本，实现了对 B 站、YouTube 等主流平台的视频链接解析与高速下载
- **攻克「防盗链」机制（403/503 报错）**：在开发过程中，通过 AI 诊断并解决了服务器限制资源访问的 Referer 防盗链问题，成功通过伪装请求头让视频不仅能解析，还能顺利下载并展示封面
- **突破免 Cookie 下载壁垒**：针对抖音等必须要求用户登录态（Cookie）才能下载的苛刻平台，规避了危险的扫码获取方案，利用 AI 挖掘网络最新的开源绕过策略，实现了真正面向公众的「万能」免登录下载
